
# 2026-03-19 更新日志

* **新增功能**

    * 新增加密相关接口：

        * [`Crypto.encrypt`](/API/Crypto?id=Cryptoencrypt)：使用指定算法对数据进行加密，支持 `ArrayBuffer` 输入。
        * [`Crypto.decrypt`](/API/Crypto?id=Cryptodecrypt)：对加密数据进行解密，需与加密参数保持一致。
        * [`Crypto.digest`](/API/Crypto?id=Cryptodigest)：计算数据摘要（哈希值），支持如 `md5`、`sha256` 等算法。

    * 新增网络数据包发送接口：

        * [`sendNetworkPacket`](/API/Packet?id=sendNetworkPacket)：发送自定义数据包到服务器。
        * [`sendLocalPacket`](/API/Packet?id=sendLocalPacket)：发送数据包至客户端本地处理逻辑。

    * 新增 `Packet` 本地发送能力：

        * [`Packet.sendToLocal`](/API/Packet?id=PacketsendToLocal)：将构建的数据包发送至本地客户端处理。

    * 新增系统线程与运行时接口：

        * [`getCurrentThreadName`](/API/System?id=getCurrentThreadName)：获取当前线程名称。
        * [`runOnUiThread`](/API/System?id=runOnUiThread)：在 UI 线程执行函数。
        * [`runOnGameThread`](/API/System?id=runOnGameThread)：在游戏线程执行函数。
        * [`getRuntimeVersion`](/API/System?id=getRuntimeVersion)：获取运行环境及游戏版本信息。

    * 新增屏幕绘制接口：

        * [`createShape`](/API/Screen?id=createShape)：创建形状并返回 ID。
        * [`updateShape`](/API/Screen?id=updateShape)：更新已有形状属性。
        * [`removeShape`](/API/Screen?id=removeShape)：移除指定形状。

* **接口优化**

    * 事件参数规范调整：

        * [`onPlayerBuildBlockEvent`](/Event/?id=onPlayerBuildBlockEvent)：参数 `side` 重命名为 `face`，语义更准确。
        * [`onPlayerDestroyBlockEvent`](/Event/?id=onPlayerDestroyBlockEvent)：参数 `side` 重命名为 `face`。
        * [`onPlayerUseItemEvent`](/Event/?id=onPlayerUseItemEvent)：参数 `side` 重命名为 `face`。

    * 新增方块破坏前事件：

        * [`onPlayerStartDestroyBlockEvent`](/Event/?id=onPlayerStartDestroyBlockEvent)：在玩家开始破坏方块时触发，支持拦截。

* **文档结构调整**

    * `summary.md`：

        * 新增 [`Crypto`](/API/Crypto) 分类入口，完善 API 目录结构。

    * 多处接口文档：

        * 统一参数命名（`side` → `face`），增强一致性。
        * 补充示例代码与说明，提升文档可读性。
