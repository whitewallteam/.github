# 2026-02-10 更新日志

* **事件接口更新**

    * [`onSendServerPacketEvent`](/Event?id=onSendServerPacketEvent)

        * 新增参数 `bin` (`arrayBuffer`)，用于获取数据包的原始二进制数据。
        * 返回值能力增强：

            * 返回 `number`：控制数据包重复发送次数。
            * 返回 `arrayBuffer`：可直接替换原始数据包内容。
        * 示例代码已同步更新，支持三参数回调形式。

    * [`onReceiveServerPacketEvent`](/Event?id=onReceiveServerPacketEvent)

        * 新增参数 `bin` (`arrayBuffer`)，表示接收到的数据包原始数据。
        * 支持返回 `arrayBuffer` 以替换原始接收数据。
        * 示例代码更新为包含 `bin` 参数的写法。

* **Packet API 文档补充**

    * [`PlayerAuthInput`](/Object?id=PlayerAuthInput) 参数扩展：

        * 新增 `interactRotation`：交互旋转（`yaw`, `pitch`）。
        * 新增 `cameraOrientation`：相机方向（`x`, `y`, `z`）。
        * 新增 `rawMove`：原始玩家移动输入数据。
        * 明确多项参数为可选项，避免与协议版本不一致导致的误用。

    * 新增写入接口：

        * [`Packet.writeString`](/API/Packet?id=Packet.writeString)
          用于向数据包写入 UTF-8 编码字符串。
        * [`Packet.writeBytes`](/API/Packet?id=Packet.writeBytes)
          用于直接写入原始 `ArrayBuffer` 字节数据，不附加长度信息。

* **对象结构修正**

    * [`PlayerAuthInput`](/Object?id=PlayerAuthInput) 对象说明表调整：

        * `inputData` 类型由 `number` 修正为 `array(number)`，准确表示 `PlayerInputFlags` 标志组。
        * 表格类型对齐与字段命名统一，提升可读性与一致性。
